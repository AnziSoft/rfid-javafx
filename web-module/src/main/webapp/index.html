People Management

This is a JEE Application to expose "person information"
You can interact with it using a REST Client. For more information:
  • Part 1: The Application
  • Part 2: Reading RFID from Java
  • Part 3: REST API and Security
  • Part 4: The Client

API

Base URL is /rest
a GET on /person return all the people on the db in json format
a POST on /person and sending a person representation in json saves it in the db if it has a on duplicated RFID
a GET on /person/rfid/{some rfid} will return a person with that rfid
a delete on /person/{some id} will delete that person
